User Flow

超级管理员 · Super Admin

跨校区最高权限：可查看与编辑全部校区的所有功能。日常以巡检 + 全局配置 + 异常处理为主，授权 校长 / 校区运营 / 市场专员 做具体执行。承担数据合规、招商加盟、组织架构调整的最终决策。

校区范围 · 全部 金额 · 全部可见 审批权限 · 全部 权限模板 · super-admin (*)

核心职责 · 超管每天 / 每周要做的事

01
🌐 跨校区数据巡检 每天 / 每周
  • 切换校区筛选器到「全部」，查看全公司汇总指标
  • 对比各校区收入、转化、续费率，识别落后校区
  • 注意：异常增长 / 异常下滑都要追查
👥 员工与角色管理 按需 / 招聘季密集
  • 员工管理（增 / 删 / 改）— 跨校区视角
  • 角色权限调整（仅超管可改 4 个模板）
  • 身份切换（mock auth）方便测试不同角色视图
💼 公司级财务审批与汇总 每周 / 月底
  • 审批校长上报的大额退款 / 报销
  • 查看跨校区总账单、工资、合同
  • 处理特殊财务事项（重大优惠、坏账）
📺 数据大屏深度分析 每周
  • 对比所有校区在 TV-wall 视图（财务 + 招生 + 教务）
  • 识别行业 / 季节趋势
  • 制定下月 / 下季度全局策略
🏢 招商加盟跟进 按需
  • 查看 franchise.html 加盟咨询表单提交
  • 跟进意向加盟商，输出合作方案
  • 新校区开业前期协调
⚙️ 全局业务设置 不常改
  • 所有校区的设置都可改（试课时段、营业时段、教室、公告）
  • 设置默认配置 (_default) 适用未单独配置的校区
  • 课程时段 / Offerings 管理
🛡️ 审计与合规检查 每月
  • 查看完整 minddo_audit_log（最多 500 条）
  • 追查异常操作（大量删除、权限变更、退款）
  • 必要时同步给法务 / HR
🚨 重大异常决策 按需
  • 校区危机处理（投诉升级、合规事件、人事冲突）
  • 系统级问题（数据异常、批量错误）
  • 支持 reseedDemoData() 重置 demo 环境（仅 demo 用）

子流程 · 跨校区数据对比

04

超管独有的"全公司视角"。每个 dashboard 模块都尊重校区筛选器：切换到「全部」时显示汇总，切到某一校区时下钻。

  1. 顶栏校区选择器：默认「全部校区（汇总）」。切换会立即重渲染当前 tab
  2. 数据中心 → 数据大屏：全公司财务 + 招生 + 教务的"指挥中心"视图
  3. 财务中心 → 订单：可按校区分组查看，识别哪个校区贡献最大
  4. 教务中心 → 学员名单：全公司学员花名册
  5. 市场中心 → CRM：全部线索，可按校区过滤分析
  6. 对比维度：本月收入 / 转化率 / 在读 / 续费率 / 出勤率，识别 top / bottom 校区
  7. 下钻调查：临时切换到某一校区视角，看具体数据；查完切回「全部」

其他独家操作

05

🛡️ 角色权限管理

仅超管可改 4 个角色模板的权限配置。

  1. 内部管理 → 角色 sub-tab
  2. 看到 4 个角色（super-admin / principal / campus-ops / campus-marketing）
  3. 编辑某角色的 permissions 数组（添加 / 删除权限字符串）
  4. 变更立即生效（hasPerm 实时读取）
  5. 谨慎操作：误删 super-admin 的 "*" 会锁死自己

👥 员工增删（跨校区）

新员工入职 / 离职管理。

  1. 内部管理 → 员工 sub-tab
  2. 看所有校区的员工（默认）或按校区筛选
  3. 新员工：填写 id / name / roleId / campus / department / 邮箱 / 电话
  4. 状态调整：active / leave / inactive
  5. 新员工默认 roleId 决定其登录后看到的界面

💎 招商加盟跟进

侧栏底部 💎 招商加盟入口。

  1. 查看 franchise.html 提交的咨询表单（minddo_franchise_inquiries）
  2. 评估意向 + 经验 + 资金
  3. 跟进沟通 / 输出合作方案
  4. 签约后协调新校区开业流程
  5. 仅超管可见，其他角色看不到此入口

🔄 身份切换（测试）

顶栏右上角"切换身份"菜单。

  1. 显示按角色排序的所有 active 员工
  2. 点击任一员工 → 系统以该员工身份重渲染
  3. 用于查看其他角色看到的界面是什么样
  4. 本质是 mock auth，写 minddo_active_ops_user
  5. 切换回 super-admin 恢复全权限

minddo_seed_version
