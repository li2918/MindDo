1. 登录与切换身份

01 / Login

当前 demo 阶段没有真正的密码登录，使用 dashboard 顶栏右上角的「切换身份」菜单选择自己的员工档案即可。系统会把所选员工 id 写入 minddo_active_ops_user。

  1. 打开 运营 Dashboard
  2. 右上角点击你的头像 / 当前用户名 → 选择「切换身份」
  3. 在弹出的列表里找到自己的姓名和岗位（一般标注为 校区运营 · campus-ops），点击。
  4. 页面会自动刷新到你权限范围内的视图。再切换其他人也是同样操作。
提示 · 校区怎么选 顶栏「当前校区看板」会默认显示你所属的校区，看板和数据都按这个校区过滤。如需暂时切到「全部校区（汇总）」查看横向对比，下拉切换即可，不影响其他人。

2. 认识界面 · 侧边栏导航

02 / Layout

左侧侧边栏是主导航。每个一级 tab 下有自己的子菜单。下表列出你作为校区运营能看到的所有 tab，以及一键直达入口：

一级 Tab 用途 直达
🏠 首页 校区运营专属首页：今日重点、待办、课程时间轴、容量预警。 打开
📥 待办 请假、改期、合同到期、试课待评估 — 统一处理面板。 打开
📣 市场中心 只读 CRM：看本校区试课线索、各渠道效果。不写入。 打开
📘 教务中心 核心工作区：学员、班级、排课、考勤。 打开
💰 财务（脱敏） 能看付费记录和续费提醒，金额显示为 $***。不能看详单 / 退款 / 工资。 打开
📈 数据中心 各维度的运营分析。财务图表对你脱敏。 打开
🛠 内部管理 本校区员工、教师列表（只读）+ 操作日志。 打开
⚙️ 业务设置 校区营业时间、教室、公告 — 只能改本校区。 打开
小技巧 · 全局搜索 在 dashboard 任意页面按 Ctrl/⌘ + K 唤起全局搜索，可以直接搜学员姓名 / 电话 / 邮箱跳到对应详情，比在列表里翻快得多。

3. 每日例行清单

03 / Daily

到岗后（约 8:30 - 9:00）

  • 打开 首页，浏览「今日重点」「待办总数」「课程时间轴」。
  • 查看 待办，处理隔夜累积的请假、改期、合同。
  • 核对今天的试课家庭名单，提前准备登记表 / 教室。
  • 跟教师确认到位情况，必要时调整排课。

课程期间（滚动）

  • 接待试课家庭：签到、引导、记录到达状态。
  • 盯首页「课程时间轴」，识别「进行中」与容量预警（满班 / 近满 / 空班）。
  • 临时事件（家长来电请假、孩子迟到）即时录入。

每节课结束

  • 到 教务中心 · 考勤 逐人录入出勤（出勤 / 缺勤 / 请假）。
  • 试课家庭：勾选「完成 / 未到」并补充评估备注。

下班前（约 18:00 - 19:00）

  • 到 首页 底部「交班记录」写当日交接（已完成、遗留、明日重点）。
  • 检查 续费提醒，把临期的家长加入今日跟进列表。
  • 清空待办收件箱（理想状态）。

4. 试课接待

04 / Trial

试课线索由市场团队录入，运营负责现场接待和评估反馈。状态机：预约 → 到达 → 试课中 → 完成 / 未到。

  1. 到 市场中心 · 试课管理 查看今日预约清单。
  2. 家长到店：在该家庭一行点「签到」按钮，状态从 booked 切到 confirmed。
  3. 开始试课：进入「试课中」（状态写入 trialStatus = "in_progress"）。
  4. 试课结束：点「完成」按钮，弹出反馈表单 — 填写表现（兴趣度、专注度、推荐等级）+ 文字评价。
  5. 提交后系统自动产生一条评估记录，状态切到 completed，市场团队会据此跟进转化。
  6. 若家庭未到：标记 noshow，原因填在备注里（如「家长当天来电请假」）。
试课评估表小窍门 表单里有「✨ AI 评估草稿」按钮。先写几句观察（孩子表现、家长反应、关注点），点击后 AI 会生成一段结构化反馈草稿。你可以一键应用到正式表单，再做手动微调，省一半时间。

5. 考勤录入

05 / Attendance

所有正式课的考勤都在教务中心录入。出勤状态包括：出勤 / 缺勤 / 请假。

  1. 打开 教务中心 · 考勤。
  2. 选择课程日期 + 班级，系统列出该班全部学员。
  3. 逐人勾选状态。请假 必须有理由（家长事先报备的可以选「家长请假」）。
  4. 保存。系统会按 ISO 时间戳写入 minddo_attendance 数组，每条记录一次性写入并审计。
  5. 如果发现某学员临时调班，先到 教务 · 排课 调整，再录今日考勤。
注意 · 不要漏录 考勤是续费转化的重要参考。一周内未录入的课次，财务侧会自动标红。建议每节课结束 15 分钟内完成录入。

6. 请假 / 改期审批

06 / Approvals

家长在自己的账号端发起请假或改期申请后，会出现在你的「待办」面板。你只能审批 请假 和 调课 这两类 — 退款 / 报销由校长处理。

  1. 打开 待办，顶部过滤选「请假改期」。
  2. 每一条卡片显示家长姓名、原课次、申请类型、备注理由。
  3. 点 ✓ 通过 或 ✗ 驳回。驳回必须填理由，会同步回传给家长。
  4. 每一次审批都自动写入 minddo_audit_log，包含操作人、对象、时间、前后状态。
  5. 处理完后这条申请从待办移除；状态变化也同步反映到对应学员的「申请历史」tab。
看到金钱相关的卡片（退款/报销）怎么办 你能看到但不能审批 — 点开是只读视图。请联系校长（principal 角色）处理。

7. 续费提醒与跟进

07 / Renewals

财务 tab 的「续费提醒」是脱敏面板：你看不到具体金额，但可以看到 哪些学员的课时 / 套餐快用完了，以及上次联系时间。这是你日常跟进的核心列表。

  1. 打开 财务 · 续费提醒。
  2. 列表按「剩余课时」升序排列，红色行 = 极临期（≤3 节）。
  3. 点学员姓名打开学员详情抽屉，看课时使用率、近期出勤、家长偏好。
  4. 沟通后，到「联系家长」按钮调出家长联系卡片（电话 / 微信 / 邮箱），不要直接打开邮箱 / 拨号 — 先看完整信息再选合适渠道。
  5. 沟通完成后，记得回 学员详情 → 变更历史 tab 留一条跟进备注（自动写审计日志）。

8. 学员信息查询

08 / Students

你可以查任意本校区学员的完整档案。从任何列表点击学员姓名即可打开详情抽屉。

抽屉里有这些 tab，都是只读 + 全审计：

Tab看什么
📋 概览基本资料、当前级别、报名套餐、剩余课时。
✅ 出勤历次出勤记录，按月分组，可以筛选异常（缺勤 / 请假）。
📝 作业课后作业完成情况 — 提交率、教师评分。
📈 成长每月成长档案曲线（兴趣度 / 编程能力 / 表达 / 独立性）。
💰 付费历次付费记录（金额脱敏为 $***）。
💬 反馈家长 / 教师反馈历史。
📨 申请该学员所有请假 / 改期申请的处理记录。
🎨 作品集项目作品 + 学期报告。
📜 变更历史该学员相关的所有操作日志（谁、什么时间、改了什么）。
快速查询 Ctrl/⌘ + K 全局搜索学员名字 → 直接打开详情抽屉，不用先进列表。

9. 交班记录

09 / Shift

每日下班前 / 接班时填写交班，确保信息无缝衔接。这是 campus-ops 独有的责任 — 校长不一定看，但下一班的同事一定要看。

  1. 到 首页 找到「交班记录」面板。
  2. 三栏结构：今日完成（X 节课、Y 个试课、Z 项审批）/ 遗留问题（待跟进的家长、设备故障等）/ 明日重点。
  3. 填完点保存 — 写入 minddo_shift_notes，所有同校区运营都看得到。
  4. 接班时第一件事就是看上一班的交班记录。
AI 帮你写 点「✨ AI 生成交班摘要」会基于今日审计日志自动总结一份草稿，你只需补充上下文细节，省 80% 时间。

10. 校区设置

10 / Settings

「业务设置」tab 是本校区配置的统一入口。你能改的仅限自己校区的：

子设置能改什么直达
🎤 试课 本校区试课时段开放规则、单时段容量。 打开
🕒 营业时间 本校区每周营业时段（周一至周日各自的开关）。 打开

12. 常见问题

12 / FAQ
看不到某个 tab 怎么办？

侧边栏会自动隐藏你权限范围外的入口。如果突然某个原来能用的 tab 不见了，检查 ① 你是不是被切换到了其他角色（顶栏右上角看当前身份）；② 是不是切到了别的校区。

金额怎么全是 $***？

这是设计如此 — campus-ops 角色的财务面板对真实金额脱敏。需要查具体数字请联系校长，或者通过「续费列表」+「学员详情」交叉核对（这些视图给你看了课时数 / 套餐名 / 续费状态等关键信息，仅金额数字脱敏）。

家长说他点了请假但我看不到？

① 到 待办 顶部过滤选「请假改期」；② 如果列表里没有，问家长是从家长端 App 提交的还是只是微信告诉你的（后者需要你手动代录）；③ 已审批过的申请会从待办移除，去对应学员详情的「申请」tab 找历史记录。

录错考勤怎么改？

直接在 考勤面板 重新选状态保存即可。系统会写入 变更 而不是覆盖 — 旧值 + 新值都保留在审计日志，谁改的、什么时间一清二楚。学员详情「变更历史」tab 也能看到。

AI 按钮（✨）安全吗？

AI 按钮只产生草稿，不会自动写入。生成结果会出现在按钮下方，需要你确认后点「应用」才生效。可以放心使用，省时间。当前 demo 环境 AI 是本地模拟（不联网），上线后会走真实 LLM 但同样是「先生成、再确认」的流程。

切换语言？

顶栏右上角的「中 / EN」按钮切换。你的偏好会被记住（写入 minddo_lang），下次打开自动恢复。

误操作了能撤销吗？

多数操作不能一键撤销，但 所有操作都有审计，可以根据 操作日志 的 diff 手动改回去。重要操作（如批量改状态）前会有确认弹窗，仔细看一眼再点。

更多问题 / 反馈渠道？

页面右下角 ✨ 客服 按钮 → 「AI 聊天」tab 直接问，常见问题秒回；复杂问题切到「联系我们」tab 看微信 / 电话 / 邮箱。或者直接 @ 校长。
