维护家长在选课页看到的班次：课程名、等级、老师、上课日和时段。所有修改会写入浏览器的运营缓存，用于展示。Maintain the class offerings parents see on the schedule page — course, level, teacher, day and time slot. Edits are saved to the ops cache (localStorage) for demo purposes.

老师可用时段Teacher Availability

点击展开 · 设置每位老师的工作时段，用于检测排课冲突Tap to expand · Set each teacher's working hours; used to flag conflicts

勾选老师在每个工作日可用，并设置起止时间。班次落在勾选范围之外会被标记为「超时排课」。 Tick the weekdays each teacher is available and set the time window. Classes outside that window are flagged as "out of hours".

批量新增班次Batch Add Class Sessions

填一次课程信息，勾选上课日并列出时段，自动生成 N 个班次（不同日期 × 不同时段的所有组合）。 Fill in the course details once, pick weekdays and list time slots — we'll generate every combination as a separate class.
